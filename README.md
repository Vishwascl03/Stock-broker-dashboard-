# Stock Broker Client Web Dashboard

##Project Overview
This project is a simple Stock Broker Client Web Dashboard that allows users to log in using their email, subscribe to selected stock ticker codes, and view real-time stock price updates without refreshing the page.

Stock prices are simulated using a random number generator and updated every second.

---

##Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Real-Time Communication: WebSocket

---

##Features
- Login using email
- Subscribe to stock ticker codes
- Supported stocks:
  - GOOG
  - TSLA
  - AMZN
  - META
  - NVDA
- Live stock price updates every second
- No page refresh required
- Supports multiple users simultaneously
- Asynchronous updates
